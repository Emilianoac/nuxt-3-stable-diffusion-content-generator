# Pixur 

A web application that allows users to generate images from text prompts using the Stability.ai API.

## Features

- 🖼️ Generate images from text prompts
- 💾 Save generated images to your devic

## Tech Stack
- Nuxt 3
- Tailwind CSS
- Stability.ai API
- Firebase
